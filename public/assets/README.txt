Put brand assets here.

Default logo file used by the site:
- /public/assets/logo.svg

Supported formats:
- logo.jpeg
- logo.png
- logo.png
- logo.webp
- logo.svg

Rendering fallback order:
1. content/site.json -> logoPath
2. /assets/logo.jpeg
3. /assets/logo.png
4. /assets/logo.png
5. /assets/logo.webp
6. /assets/logo.svg

To replace with your own logo:
1. Keep the same file name (logo.svg), or
2. Add your file (e.g., logo.png) and update content/site.json -> logoPath.
